# NET（Cloudflare）— Momentum 研究报告
> 更新日期：2026年3月

---

## 1. 一句话定位

**"AI时代的互联网神经系统——唯一能将安全、网络、边缘算力三合一的全球连接云龙头。"**

Cloudflare 起家于 CDN / DDoS 防护，已演进为覆盖网络安全（SASE / Zero Trust）、应用性能、开发者平台（Workers AI）的统一"连接云（Connectivity Cloud）"。在 AI Agent 大规模落地的背景下，Cloudflare 的全球边缘网络正从可选基础设施转变为必要基础设施。

- **所属板块**：云计算 / 网络安全 / AI基础设施  
- **子分类**：网络安全-平台化 / AI-边缘基础设施

---

## 2. 主题受益深度：直接受益 ✅

Cloudflare 是 AI 基础设施浪潮的**直接受益者**，而非概念蹭热度：

| 受益维度 | 具体逻辑 |
|---|---|
| AI Agent 流量爆炸 | AI Agent 产生的 API 请求量远超人类用户，直接拉升 Cloudflare 网络吞吐量与安全需求 |
| Workers AI 平台货币化 | 开发者在 Cloudflare 边缘直接运行推理，已成为 FY2026 超预期的核心驱动 |
| AI 公司选 Cloudflare 为主基础设施商 | Q4 财报披露多家头部 AI 公司（未具名）将 Cloudflare 选为**首要基础设施提供商**，而非 AWS/Azure |
| AI 治理 / 安全合规 | 2025年 SEC AI 治理披露新规落地，Cloudflare 统一仪表盘大幅降低合规成本，直接受益企业客户升级需求 |

管理层将当前阶段定义为互联网的"fundamental re-platforming"，AI 是这一重塑的核心驱动力。

## 3. 板块内比较：相对竞争优势

**主要对标**：CrowdStrike（CRWD）、Zscaler（ZS）、Palo Alto Networks（PANW）

| 维度 | Cloudflare NET | CRWD | ZS | PANW |
|---|---|---|---|---|
| 产品完整性 | 安全+网络+算力三合一 | 纯终端安全 | 纯 SASE/Zero Trust | 平台化但重 on-prem |
| AI 原生能力 | Workers AI 边缘推理，最早落地 | 有 AI 功能但非平台核心 | 较弱 | Charlotte AI 有限 |
| 开发者生态 | 极强（Pages/R2/Durable Objects） | 几乎无 | 无 | 无 |
| 大客户增速 | >$100k 客户增 23%，$1M+ 客户创历史新高 | 强但增速放缓 | 放缓 | 稳健 |
| 超大客户中立性 | 被 AI 公司选为"中立替代超大云"，差异化明显 | 无此定位 | 无 | 无 |

**核心优势**：Cloudflare 是板块内**唯一**同时具备"安全平台 + AI 算力边缘 + 开发者生态"三位一体的公司。AI 公司优先选择 Cloudflare 而非 AWS/Azure，正是因为其云中立性与统一架构。

## 6. 风险雷点

### ⚠️ 风险一：估值过高 × 持续亏损 = 宏观敏感性极强（首要风险）
Cloudflare 当前 PS 倍数仍处高位，且净利润为负（Q4 净亏损 $12.08M）。一旦市场风险偏好收缩（美联储表态偏鹰、科技股情绪降温），哪怕基本面不变，股价也可能跌 20-30%。2月23日已出现单日 -7.8% 的回调，验证了这一脆弱性。

**监控指标**：10年期美债收益率走势、科技股 VIX 分项指标。

### ⚠️ 风险二：用量型收入波动性（季度执行风险）
CFO 已明确指出，"池式合同（pool of funds）"模式虽然深化客户关系，但 AI 工作负载用量不可预测，导致单季度收入确认具有高波动性。若某季度 AI 用量不及预期，即便年度指引不变，市场仍可能以"miss"定价。

**监控指标**：每季财报中 usage-based revenue 占比、大客户净扩张率（NRR）。
